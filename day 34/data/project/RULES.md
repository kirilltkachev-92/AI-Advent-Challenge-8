# Правила проекта «Инфопанель»

1. Все публичные функции обязаны иметь KDoc-комментарий (`/** ... */`).
2. Все сетевые вызовы идут только через `HttpFetcher.get` — никаких
   прямых обращений к `HttpClient` вне `HttpFetcher.kt`.
3. Каждый публичный API-объект (`*Api`) должен быть описан в `docs/README.md`.
4. В README команда запуска должна совпадать с реальной: `kotlinc src -include-runtime -d app.jar && java -jar app.jar`.
