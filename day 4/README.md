# День 4 — температура

Десктопное приложение на Kotlin Compose сравнивает ответы DeepSeek на **один и тот же запрос** при разных значениях `temperature`.

Диапазон: **от 0 до 2 с шагом 0.3** — `0`, `0.3`, `0.6`, `0.9`, `1.2`, `1.5`, `1.8`, `2.0`.

## Режимы

- **Последовательно все температуры + сравнение** — по очереди выполняются запросы для каждого значения, затем API формирует сравнение по **точности**, **креативности** и **разнообразию**, тренд по шкале temperature и рекомендации по выбору значения.
- **Одна выбранная температура** — один запрос с выбранным значением из списка.

**Скорость вывода:** «Медленно» — ответы печатаются посимвольно; «Быстро» — текст появляется сразу целиком.

## API-ключ

Ключ берётся из (по порядку):

- переменной окружения `DEEPSEEK_API_KEY`
- файла `.env` в папке `day 4`

## Запуск

```bash
cd "day 4"
chmod +x run.sh
./run.sh
```

Или:

```bash
./gradlew run
```

Требуется Java 17.
