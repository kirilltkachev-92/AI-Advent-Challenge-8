## AI Advent Challenge #8 — Итоговый релиз

Этот репозиторий — результат 35-дневного марафона по созданию Kotlin-приложений вокруг больших языковых моделей (LLM). Каждый день — отдельное приложение, которое шаг за шагом раскрывает возможности интеграции LLM в реальные проекты.

### Темы дней

- **Основы работы с LLM** (дни 1–5): формат ответа, рассуждения, температура, версии моделей.
- **Агенты и контекст** (дни 6–12): первый агент, сохранение контекста, работа с токенами, сжатие истории, стратегии управления контекстом, модель памяти, персонализация.
- **Агенты с состоянием и планирование** (дни 13–15): кодинг-агент с конечным автоматом, инварианты, контролируемые переходы.
- **MCP и оркестрация** (дни 16–20): подключение MCP, инструменты, планировщик, композиция, оркестрация.
- **RAG (Retrieval-Augmented Generation)** (дни 21–25): индексация, запросы, реранкинг, цитаты, мини-чат с RAG и памятью.
- **Локальные LLM** (дни 26–30): запуск, интеграция, RAG, оптимизация, приватный сервис.
- **Прикладные ассистенты** (дни 31–35): ассистент разработчика, ревью кода, поддержка пользователей, работа с файлами, AI-конвейер подготовки релиза.

### Самые интересные дни
- **День 6** — первый агент: закладывает основу для всех последующих агентских решений.
- **День 13** — кодинг-агент с конечным автоматом и роем агентов: сложная композиция.
- **День 16** — подключение MCP: открывает мир протоколов и инструментов.
- **День 22** — первый RAG-запрос: ключевой момент для работы с внешними данными.
- **День 25** — мини-чат с RAG + памятью: production-like решение.
- **День 35** — AI-конвейер подготовки релиза: практическое применение всего изученного.

### Как запускать

В каждом каталоге `day N` есть README с описанием и `run.sh` для запуска. Требуется JDK 17 и API-ключ DeepSeek (установите переменную окружения `DEEPSEEK_API_KEY`).
