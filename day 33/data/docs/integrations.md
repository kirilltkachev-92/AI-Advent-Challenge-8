# Интеграции и API

## Telegram-бот

Бот @PlanerkaBot присылает напоминания о задачах и утреннюю сводку планёрки.

Подключение: «Настройки профиля → Интеграции → Telegram» → кнопка
«Получить код» → отправить боту команду `/link <код>`. Код действует 15 минут.

Если напоминания не приходят:

1. Проверить, что интеграция в статусе «Подключено» (зелёная точка).
2. Проверить, что бот не остановлен: в Telegram открыть чат с ботом
   и нажать «Запустить» (после блокировки бота связку нужно пересоздать).
3. Проверить настройки уведомлений: «Настройки профиля → Уведомления» —
   напоминания о задачах и сводка планёрки включаются отдельно.
4. Напоминания приходят только по задачам, где вы исполнитель,
   и только в рабочие часы, заданные в профиле (по умолчанию 9:00–19:00).

## Google Calendar

Задачи со сроком появляются в отдельном календаре «Планёрка».
Синхронизация — раз в 15 минут, односторонняя (из Планёрки в календарь).
Перенос события в календаре не меняет срок задачи.

## API и вебхуки (тариф Business)

Токен выпускается в «Настройки команды → API». Базовый адрес —
api.planerka.app, версия v1. Токен передаётся в заголовке Authorization.

Лимит — **60 запросов в минуту на токен**. При превышении API отвечает
HTTP 429 с заголовком Retry-After (секунды до сброса окна). Правильная
обработка — пауза на Retry-After и повтор; при регулярных 429 стоит
кешировать ответы и объединять запросы в батчи.

Вебхуки шлют POST при событиях задач (создана, изменена, закрыта);
подпись — HMAC-SHA256 в заголовке X-Planerka-Signature.
