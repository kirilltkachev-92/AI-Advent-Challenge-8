## AI Advent Challenge #8 — Итоговый релиз

Этот репозиторий — результат 35-дневного марафона по созданию Kotlin-приложений вокруг больших языковых моделей. Каждый день — новая задача, от простых вызовов API до полноценных агентов и конвейеров.

### Темы дней

- **Основы работы с LLM** (дни 1–5): формат ответа, рассуждения, температура, версии моделей.
- **Протоколы и интеграции** (дни 6–10): первый агент, сохранение контекста, токены, сжатие истории, стратегии управления контекстом.
- **Память и персонализация** (дни 11–12): модель памяти ассистента, персонализация.
- **Агенты и конечные автоматы** (дни 13–15): кодинг-агент, инварианты, контролируемые переходы.
- **MCP (Model Context Protocol)** (дни 16–20): подключение, инструменты, планировщик, композиция, оркестрация.
- **RAG (Retrieval-Augmented Generation)** (дни 21–25): индексация, запросы, реранкинг, цитаты, мини-чат с RAG.
- **Локальные LLM** (дни 26–30): запуск, интеграция, RAG, оптимизация, приватный сервис.
- **Прикладные ассистенты** (дни 31–35): ассистент разработчика, ревью кода, поддержка пользователей, работа с файлами, AI-конвейер подготовки релиза.

### Самые интересные дни

- **День 6** — первый агент: зачатки автономного поведения.
- **День 13** — кодинг-агент с конечным автоматом и роем агентов.
- **День 16** — подключение MCP: открывает мир протоколов.
- **День 22** — первый RAG-запрос: поиск по документам.
- **День 26** — запуск локальной LLM: полный контроль.
- **День 35** — AI-конвейер подготовки релиза: венец марафона.

### Как запускать

В каждом каталоге `day N` есть `README.md` с описанием и `run.sh` для запуска. Требуется JDK 17 и API-ключ DeepSeek (установите переменную окружения `DEEPSEEK_API_KEY`).
