## AI Advent Challenge #8 — Итоговый релиз

Этот репозиторий — результат 35-дневного марафона по созданию Kotlin-приложений вокруг языковых моделей. Каждый день — отдельное приложение, которое шаг за шагом раскрывает возможности интеграции LLM в реальные проекты.

### Темы дней

- **Основы работы с LLM** (дни 2–5): формат ответа, способы рассуждения, температура, версии моделей.
- **Агенты и контекст** (дни 6–12): первый агент, сохранение контекста, работа с токенами, сжатие истории, стратегии управления контекстом, модель памяти, персонализация.
- **Управление состоянием и агенты** (дни 13–15): кодинг-агент с конечным автоматом, инварианты и ограничения состояния, контролируемые переходы.
- **MCP-протокол и интеграции** (дни 16–20): подключение MCP, первый инструмент, планировщик, композиция, оркестрация.
- **RAG (Retrieval-Augmented Generation)** (дни 21–25): индексация документов, первый RAG-запрос, реранкинг, цитаты и анти-галлюцинации, мини-чат с RAG и памятью.
- **Локальные LLM** (дни 26–30): запуск локальной модели, интеграция, RAG, оптимизация, приватный сервис.
- **Прикладные ассистенты** (дни 31–34): ассистент разработчика, автоматизация ревью кода, поддержка пользователей, работа с файлами проекта.

### Самые интересные дни

- **День 6** — первый агент: закладывает основы агентного подхода.
- **День 13** — кодинг-агент с конечным автоматом и роем агентов.
- **День 16** — подключение MCP: открывает мир протоколов.
- **День 22** — первый RAG-запрос: ключевой шаг к работе с документами.
- **День 25** — мини-чат с RAG + памятью: production-like решение.
- **День 31** — ассистент разработчика: практическое применение.

### Как запускать

В каждом каталоге `day N` есть README с описанием и скрипт `run.sh`. Для запуска требуется JDK 17 и API-ключ DeepSeek (указан в переменной окружения `DEEPSEEK_API_KEY`).
