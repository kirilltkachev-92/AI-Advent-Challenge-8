# День 6 — первый агент

Десктопный чат на Kotlin Compose с **агентом**, который инкапсулирует работу с LLM.

## Архитектура

```
UI (Main.kt)  →  ChatAgent  →  DeepSeekClient  →  DeepSeek API
```

- **ChatAgent** — отдельная сущность: хранит историю диалога, принимает сообщение пользователя, отправляет **всю** историю в API, сохраняет ответ ассистента.
- **DeepSeekClient** — HTTP-клиент (запрос/ответ JSON).
- **Main.kt** — только отображение и ввод; логика запроса не дублируется в UI.

История **накапливается**: каждый новый запрос видит предыдущие реплики user/assistant (плюс system prompt).

## API-ключ

Переменная окружения `DEEPSEEK_API_KEY` или файл `.env` (см. `.env.example`).  

## Запуск

```bash
cd "day 6"
chmod +x run.sh
./run.sh
```

Или:

```bash
./gradlew run
```

Требуется Java 17.

## Команды в интерфейсе

- **Отправить** — передать сообщение агенту (с учётом истории).
- **Очистить историю** — сброс диалога (system prompt остаётся).
