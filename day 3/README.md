# День 3 — разные способы рассуждения

Десктопное приложение на Kotlin Compose сравнивает четыре подхода к решению одной задачи через DeepSeek API.

## Способы рассуждения

1. **Прямой ответ** — задача отправляется без дополнительных инструкций.
2. **Пошагово** — в промпт добавлено «решай пошагово».
3. **Мета-промпт** — модель сначала составляет промпт для решения, затем решает задачу по нему.
4. **Панель экспертов** — теоретик, алгоритмист, верификатор и сводный итог.

В режиме **«Последовательно все 4 способа + сравнение»** все пункты выполняются по очереди, затем API формирует сравнение: отличаются ли ответы и какой способ точнее.

**Скорость вывода:** «Медленно» — ответы печатаются посимвольно; «Быстро» — промпты и ответы появляются сразу целиком.

## API-ключ

Ключ берётся из (по порядку):

- переменной окружения `DEEPSEEK_API_KEY`
- файла `.env` в папке `day 3`

## Запуск

```bash
cd "day 3"
chmod +x run.sh
./run.sh
```

Или:

```bash
./gradlew run
```

Требуется Java 17.
