Конечно! Вот пример на Kotlin функции `isPalindrome`:

```kotlin
fun isPalindrome(s: String): Boolean {
    // Используем Java's StringBuilder для очистки строки
    val sb = StringBuilder(s)
    
    // Переводим все символы в нижний регистр
    sb.reverse()
    
    // Проверяем, равны ли исходная и пересчитанная строки
    return sb.toString() == s
}

// Пример использования
fun main() {
    val testString = "A man, a plan, a canal: Panama"
    println("Is the string '$testString' a palindrome? $isPalindrome(testString)")
}
```

Эта функция `isPalindrome` принимает строку `s`, преобразует все символы в нижний регистр и затем сравнивает исходную строку с пересчитанной строкой. Если они равны, функция вернет `true`, иначе `false`.
